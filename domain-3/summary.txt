A DEFAULT GATEWAY AND DEFAULT ROUTER ARE SYNONYMS.

5 STEPS IT TAKES FOR A ROUTER TO FORWARD RECEIVED DATA FRAME.
-1A - THE FRAME HAS NO ERRORS
-1B - THE FRAME DESTINATION DATA-LINK IS THE ROUTERS ADDRESS
-2 - DE-ENCAPSULATE THE PACKET FROM DATA-LINK
-3 - MAKE A ROUTING DECISION
-4 - ENCAPSULATE THE PACKET
-5 - TRANSMIT THE FRAME

MATCHING DECISION
-SUBNET ID
-MASK

FORWARDING DECISION
-NEXT ROUTER
-OUT INTERFACE

PREFIX MEANS SUBNET ID

CONFIGURING STATIC ROUTES
A STATIC NETWORK ROUTE DEFINES EITHER A SUBNET OR AN ENTIRE CLASS A, B OR C NETWORK IN THE IP ROUTE COMMAND.

-A DEFAULT ROUTE DEFINES THE SET OF ALL DESTINATION IP ADDRESS

-A HOST ROUTE DEFINES A SINGLE IP ADDRESS

USING LOCAL INTERFACE AS OUT GOING FORWARDING METHOD FORCES THE USE OF PROXY ARP

TROUBLESHOOTING STATIC ROUTE
-THE ROUTE IS IN THE ROUTING TABLE BUT IS INCORRECT
-THE ROUTE IS NOT IN THE ROUTING TABLE
-THE ROUTE IS IN THE ROUTING TABLE AND IS CORRECT, BUT THE PACKETS DO NOT ARRIVE AT THE DESTINATION HOST

USING PERMANENT KEYWORD INSTRUCT THE ROUTER TO INCLUDE ROUTE NO MATTER WHAT.

Interior and Exterior Routing Protocols

-IGP - A ROUTING PROTOCOL THAT WAS DESIGN AND INTENDED FOR USE INSIDE A SINGLE AUTONOMOUS SYSTEMS (AS)

-EGP - A ROUTING PROTOCOL THAT WAS DESIGNED AND INTENDED FOR USE BETWEEN DIFFERENT AUTONOMOUS SYSTEMS.

IGP Routing Protocol Algorithms
-DISTANCE VECTOR - BELLMAN-FORD
-ADVANCED DISTANCE VECTOR - CALLED BALANCED HYBRID
-LINK-STATE

Metrics
-RIPV2 - HOP COUNT
-OSPF - END-END INTERFACE COST
-EIGRP - BANDWIDTH AND DELAY

Other IGP Comparisons
