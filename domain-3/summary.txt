ROUTER INTERFACES
ROUTER SUPPORTS VARIETY OF OTHER TYPES OF INTERFACES, INCLUDING SERIAL INTERFACES, CABLE TV, 4G/5G WIRELESS, AND OTHERS NOT MENTIONED IN THIS BOOK.

FOR EXAMPLE, A 10-MBPS-ONLY ETHERNET INTERFACE WOULD BE CONFIGURED WITH THE INTERFACE ETHERNET NUMBER CONFIGURATION COMMAND, 10/100 INTERFACE WITH THE INTERFACE FAST ETHERNET NUMBER COMMAND, AND A 10/100/100 INTERFACE GIGABIT ETHERNET NUMBER COMMAND.

SERIAL LINK HAS HIGH-LEVEL DATA LINK CONTROL (HDLC DEFAULT) OR POINT-TO-POINT PROTOCOL (PPP)

TWO MOST COMMON COMMAND TO DISPLAY STATUS ARE:
SHOW IP INTERFACE BRIEF
SHOW INTERFACES

INTERFACE STATUS CODES
STATUS - LAYER 1 (PHYSICAL LAYER)
PROTOCOL - LAYER 2 (DATA LAYER)

SHOW COMMAND FOR VERIFICATION
SHOW IP INTERFACE BRIEF
SHOW INTERFACE TYPE NUMBER
SHOW PROTOCOL TYPE NUMBER
SHOW INTERFACE TYPE NUMBER CONTROLLER
